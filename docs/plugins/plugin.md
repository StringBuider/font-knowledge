> 简介: form-create 是一个可以通过 JSON 生成具有动态渲染、数据收集、验证和提交功能的表单生成组件。支持3个UI框架，并且支持生成任何  Vue 组件。内置20种常用表单组件和自定义组件，再复杂的表单都可以轻松搞定。<br>
